# youtube-downloader

This is a simple youtube downloader webapp that uses the yt-dlp to download videos from youtube.

## Development

### Requirements

- Go
- npm
- Node.js

Install templ:

```bash
go install github.com/a-h/templ/cmd/templ@latest
```

Install node dependencies:

```bash
npm install
```

Run the development server:

```bash
make dev
```
